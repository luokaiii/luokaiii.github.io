# 样式设计

在 React Native 中，仍然使用 JavaScript 来写样式。所有核心组件都接受名为 style 的属性。

在实际开发中，推荐使用 `StyleSheet.create` 来集中定义组件的样式。

## 二、高度宽度

在 React Native 中，元素是没有px、em之类的宽度的，而是以像素作为宽高。
