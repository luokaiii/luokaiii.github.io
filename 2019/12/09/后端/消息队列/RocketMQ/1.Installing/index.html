下载与安装

一、Docker Images

执行以下命令，将 RabbitMQ 运行在 Docker 中：

docker run -it --rm --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3-management

windows环境下，可能会提示需要加上 “winpty” 前缀，照做即可。
